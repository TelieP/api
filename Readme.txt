Cet API a pour role d'aider au calcul des indemnités kilolométrique.
Pour effectuer le calcul , l'utilisateur doit renseigner certaines informations respectant un certain nombre de critiques:

1. La puissance fiscale doit etre supérieure à 3 ;
2. la distance suppérieure à 0 ;

A° FORMULES DE CALCULE

A.1.  "puissance fiscale" = 3
 indemnités Kilometriques = distance * 0.529
